SMF Package Software Development Kit

Working on packages?

This software development kit, for package and mod developers, is here to help people quickly begin to create packages and modifications for SMF. More thorough and advanced information can always be found on Simple Machines' Mod site.

You can find the following information in this file:
  • » Information contained in this package.
  • » Getting additional questions answered.

What's available in this SDK?

There are a number of folders in this archive, along with this readme file. The folders and files are as follows:
modifications
Examples and a guide to developing modifications for SMF.
server
Information on how you can set up your own package server to put mods up for download!
test_modification.php
A tool to test boardmod or xml style modification files with extra debugging information.
Please click the links to obtain more information about each section.

Getting more information

More information can always be found at the Simple Machines Community forum, where you can ask questions, exchange ideas, comment on this SDK, and many other things.

You can also find information on the Mod site, as well as Customize section on the community forum.
